Sobre mí

Soy un desarrollador de software apasionado por la tecnología y la creación de soluciones robustas con código limpio, mantenible y eficiente. Me especializo en el desarrollo backend y disfruto trabajar con múltiples lenguajes y paradigmas de programación.

Tengo experiencia construyendo APIs REST de alto rendimiento con Go, desarrollando aplicaciones empresariales con Java y Spring Boot, y creando plataformas full stack con TypeScript y Node.js. También trabajo con Python para automatización, web scraping y procesamiento de datos.

Me motiva resolver problemas complejos, diseñar arquitecturas escalables y aprender constantemente nuevas herramientas y metodologías para entregar software de calidad. Busco oportunidades donde pueda aportar valor con mis habilidades técnicas y seguir creciendo como profesional.
